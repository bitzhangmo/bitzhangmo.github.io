ETF, 债券, 基金, 期货交易, 股票

关键词 : 股票,债券,基金,ETF,期货交易

证券

定义: 各类记载并代表一定权利的法律凭证, 它是多种经济权益凭证的统称, 用来证明持券人有权按其券面所载内容取的应有权益的书面证明。

有价证券:

定义: 指标有票面金额, 用于证明持有人或该证券指定的特定主体对特定财产拥有所有权或债权的凭证。

有价证券代表了一定量的财产权利， 持有人可凭借这种权利获得一定量的商品和货币，或是取得利息、股息等收入。但证券本身没有价值，只是虚拟资本的一种形式。

特征：

  1. 收益性：投资者的目的是得到一定的收益
  2. 风险性：证券收益的不稳定性
  3. 流动性：持有人可以依法转让收回自己的本金
  4. 期限性：债券一般都有明确的还本付息期限，股票则没有期限。

类型：

  1. 商品证券（提货单、运货单）
  2. 货币证券（商业本票、银行汇票、支票等）
  3. 资本证券（金融投资活动产生的有价证券）

证券市场：

定义：价值，财产权利，风险直接交换的场所。

基本功能：

  1. 筹资——投资功能
  2. 定价功能
  3. 资本配置功能
    %% %%
    参与者:
  4. 证券发行人：为筹措资金而发行债券、股票等证券的发行主体
  5. 证券投资人：
  6. 证券市场中介机构
  7. 自律性组织：证券交易所和证券业协会
  8. 证券监管机构：中国证监会

证券价格指数:

  1. 股价平均数: 以绝对数的形式来反映股票市场总体价格水平。
  2. 股价指数: 将计算期的股价或市值与某一基期的股价或市值相比较的相对变化值。

主要价格指数：

  1. 沪深300：2004年12月31日，基点为1000点

    股票：

概念：是一种有价证券，是股份有限公司签发的证明股东所持股份的凭证。

性质：

  1. 股票是有价证券
  2. 股票是要式证券
  3. 股票是证权证券：证券是权利的一种物化的外在形式，代表的是股东权利
  4. 股票是资本证券
  5. 股票是综合权利证券

特征：

  1. 收益性：能否带来经济收益
  2. 风险性：收益不确定
  3. 流动性：能否通过转让来变现
  4. 永久性：是否规定了退还本金的期限
  5. 参与性：有无权利参与发行公司的管理

价值：股票本身没有价值

价格：

  1. 股票的市场价格
  2. 股票的理论价格：以一定利率计算出来的未来收益的现值 = 预期股息/必要收益率

影响股价变动的基本因素：

  1. 公司经营情况
    1. 公司治理水平和管理层质量
    2. 公司竞争力
    3. 财务状况：净资产收益率（ROE）= 净利润/净资产 = 净利润/销售收入×销售收入/总资产×总资产/净资产 = 销售利润率×资产周转率×杠杆比率
    4. 公司改组或合并
  2. 行业与部门因素
    1. 行业分类
    2. 行业分析因素
    3. 行业生命周期
  3. 宏观经济与政策因素
    1. 经济增长
    2. 经济周期循环
    3. 货币政策
    4. 财政政策
    5. 市场利率
    6. 通货膨胀
    7. 汇率变化
    8. 国际收支状况
  4. 其他因素：
    1. 政治及其他不可抗力
    2. 心理因素
    3. 稳定市场的政策与制度安排
    4. 人为操纵因素

普通股票与优先股票:

  1. 普通股票:
    1. 公司重大决策参与权
    2. 公司资产收益权和剩余资产分配权
    3. 其他权利(了解经营状况、转让股票、优先认股)
  2. 优先股票：
    1. 股息率固定
    2. 股息分派优先
    3. 剩余资产分配优先
    4. 一般无表决权

我国股票的类型：

  1. 根据投资主体分类：
    1. 国家股：有权代表国家投资的部门或机构以国有资产向公司投资形成的股份
    2. 法人股：企业法人或具有法人资格的单位与团体以其依法可支配的资产投入公司形成的股份
    3. 社会公众股：社会公众以其拥有的财产投入公司时可上市流通的股份
    4. 外资股：股份公司向外国投资者和港澳台地区投资者发行的股票
  2. 流通受限分类：
    1. 已完成股权分置改革的公司
      1. 有限售条件股份：国家持股、国有法人持股、内部职工股、其他内资持股
      2. 无限售条件股份：人民币普通股（A股），境内上市外资股（B股），境外上市外资股（H股）
    2. 未完成股权分置改革的公司
      1. 未上市流通股份：发起人股份、募集法人股份、内部职工股、优先股及其他
      2. 已上市流通股份：已在证券交易所上市交易的股份

相关概念：

H股：注册在中国内地，上市地在中国香港的外资股
S股：注册地在中国内地，上市地在新加坡的外资股
N股：~，上市地在美国纽约的外资股
L股：~，上市地在英国伦敦的外资股
红筹股：在中国境外注册，在香港上市但主要业务在中国内地或大部分股东权益来自中国内地的股票
增发：因业务发展需要增加资本额而发行新股
配股：面向原有股东，按持股数量的一定比例增发新股
转增股本：将原来属于股东权益的资本公积转化为实收资本
股份回购：上市公司利用自有资金，在公开市场上买回发行在外的股票
股票分割：将一股股票均等的拆成若干股

债券：

含义：一种有价证券，是社会各类经济主体为筹集资金而向债券投资者出具的、承诺按一定利率定期支付利息并到期偿还本金的债权债务凭证

特征：

  1. 偿还性：有固定的偿还期限
  2. 流动性：可以根据市场灵活转让债券
  3. 安全性：收益相当稳定
  4. 收益性：能带来一定收入

类型:

票面要素: (其实也就是欠条的要素吧)

  1. 票面价值
  2. 偿还期限
  3. 票面利率
  4. 发行者名称

基本性质:

  1. 有价证券
  2. 虚拟资本
  3. 债权关系的体现

股票与债券的异同点:

证券投资基金:

含义: 通过公开发行基金份额募集资金，由基金托管人托管，由基金管理人管理和运作基金，为基金份额持有人的利益，以资产组合方式进行证券投资活动的基金。

特点：（此处应该打个问号，只能说愿望很好）

  1. 集合投资
  2. 分散风险
  3. 专业理财

类型：

与股票、债券的区别：

契约型基金与公司型基金:

封闭式基金与开放式基金:

交易所交易的开放式基金:

  1. ETF (Exchange Traded Funds): 交易型开放式指数基金，申购是用一揽子股票换取ETF份额，赎回时也是换回一揽子股票而不是现金。
  2. LOF(Listed Open-Ended Funds): 可以同时在场外市场进行收购、赎回

基金资产估值:

  1. 资产总值：基金所拥有的各类证券的价值、银行存款本息、基金应收的申购基金款以及其他投资所形成的价值总和
  2. 资产净值：基金资产总值-负债

金融衍生工具：

市场交易的组织形态：

  1. 现货交易：以现款买现货方式进行交易
  2. 远期交易：双方约定在未来某时刻按照现在确定的价格进行交易
  3. 期货交易：在交易所进行的标准化的远期交易

金融期货:

金融期货是期货交易的一种。
期货交易是指交易双方在集中的交易所市场以公开竞价方式所进行的期货合约的交易。
期货合约是由交易双方订立的、约定在未来某日期按成交时约定的价格交割一定数量的某种商品的标准化协议。

期货合约>期货交易>金融期货

种类:

  1. 外汇期货：以外汇为基础工具的期货合约，主要用于规避外汇风险。
  2. 利率期货：一定数量的与利率相关的某种金融工具，为了规避利率风险
  3. 股权类期货：以股票、股票组合或股票价格指数为基础资产的期货合约。

功能：

  1. 套期保值功能：通过在现货市场与期货市场建立相反的头寸（款项），从而锁定未来现金流的交易行为。
  2. 价格发现功能：通过集中竞价形成期货价格
  3. 投机功能：利用预期建立多/空头，维持市场流动性
  4. 套利功能：忽略交易费用的差异，同一商品只能有一个价格

金融互换交易：

两个或以上的当事人按共同商定的条件，在约定的时间内定期交换现金流的金融交易。从交易结构上看，可以将互换交易视为一系列远期交易的组合。

用途：改变交易者资产或负债的风险结构，从而规避风险。

金融期权：

期权又称选择权，指其持有者能再规定的期限内按交易双方商定的价格购买或出售一定数量的基础工具的权利。期权交易就是对这种选择权的买卖。

期权交易实际上是一种权利的单方面有偿让渡。期权的买方以支付一定数量的期权费为代价而拥有了这种权利，但不承担必须买进或卖出的义务；期权的卖方则在收取了一定数量的期权费后，在一定期限内必须无条件服从买方的选择并履行成交时的允诺。

期权卖方相当于中介?

权证:

权证是基础证券发行人或其以外的第三人发行的，预定持有人在规定期间内或特定到期日，有权按约定价格向发行人购买或出售标的证券，或以现金结算方式收取结算差价的有价证券。

可转换债券：

持有者可以在一定时期内按一定比例或价格将之转换成一定数量的另一种证券的债券。

双重选择权：投资者可自行选择是否转股，并为此承担转债利率较低的机会成本；发行人拥有是否实施赎回条款的选择权，并为此要支付比没有赎回条款的转债更高的利率。

存托凭证：

一国证券市场流通的代表外国公司有价证券的可转让凭证。

资产证券化：

以特定资产组合或特定现金流为支持，发行可交易证券的一种融资形式。

将流动性较低的资产(如银行贷款，应收账款、房地产等)转化为具有较高流动性的可交易证券，提高了基础资产的流动性，便于投资者进行投资；还可以改变发起人的资产结构，改善资产质量，较快发起人资金周转。

证券投资收益与风险:

股票收益:

  1. 股息：股票持有者依据所持股票从发行公司分取的赢利。
  2. 资本利得：股票买入价与卖出价之间的差额。
  3. 公积金转增股本：送股，资金来自公司提取的公积金

债券收益：

  1. 债息：取决于债券的票面利率和付息方式
  2. 资本利得：债券买入价与卖出价或买入价与到期偿还额之间的差额
  3. 再投资收益：投资债券所获现金流量再投资的利息收入

风险：

  1. 系统风险
    1. 政策风险
    2. 经济周期波动风险
    3. 利率风险
    4. 购买力风险
  2. 非系统风险：只对某个行业或个别公司的证券产生影响的风险
    1. 信用风险
    2. 经营风险
    3. 财务风险

承担风险是获取收益的前提，收益是风险的成本与报酬

预期收益率 = 无风险收益率+风险补偿
